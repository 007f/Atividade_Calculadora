# Calculadora React Native WEB/Mobile

Objetivo Geral:
Desenvolver uma calculadora basica touchscreen para aplicações Mobile e Web, utilizando react native.  
Regras de Negocio (RN):  
RN-01 - Disponibilidade de teclado numérico: a calculadora deve disponibilizar um  
teclado numérico na tela composto de botões para o usuário selecionar via clique/toque.  
RN-02 - Apresentação de Seleção: Todo número selecionado deve aparecer em uma tela  
específica concatenado com valores anteriormente selecionados.  
RN-03 - Operações matemáticas: A calculadora deve disponibilizar as 4 operações  
matemáticas básicas (+, -, *, /) e a operação de Raiz Quadrada para o usuário.  
RN-04 - Solicitação de cálculo: A calculadora deve disponibilizar um botão de calculo para  
o usuário efetuar a operação.  
RN-05 - Apresentação de Resultado: A calculadora deve apresentar na mesma tela de  
Seleção o Resultado do cálculo após a solicitação.  
RN-06 - Solicitação para apagar: A calculadora deve apresentar um botão que apague o  
último número selecionado da tela de apresentação e todo o valor de Resultado.  

Requisitos Funcionais (RF)  
● RF-01: O sistema deve exibir um teclado numérico interativo na tela, permitindo que  
o usuário selecione os números por meio de toques ou cliques.  
● RF-02: O sistema deve conter um display (tela de apresentação) para exibir os  
números que o usuário selecionar.  
● RF-03: O sistema deve concatenar e exibir os números digitados sequencialmente
na tela de apresentação.  
● RF-04: O sistema deve fornecer botões visíveis para as quatro operações
matemáticas básicas: adição (+), subtração (-), multiplicação (*) e divisão (/).  
● RF-05: O sistema deve incluir um botão de cálculo (como um botão de igualdade
"=") que o usuário possa acionar para processar a operação desejada.  
● RF-06: O sistema deve exibir o resultado final da operação matemática diretamente  
na tela de apresentação, a mesma utilizada para mostrar a seleção dos números.  
● RF-07: O sistema deve dispor de um botão de exclusão que permita ao usuário  
apagar apenas o último número selecionado na tela de apresentação.  
● RF-08: O acionamento do botão de exclusão deve limpar simultaneamente qualquer  
valor de resultado que esteja sendo exibido.  
● RF-09: O sistema deve fornecer um botão visivel para a operação matematica  
"Raiz Quadrada" representada pela letra "R".  
● RF-10: A operação matematica "Raiz Quadrada" deve ser acionada sobre o valor que  
que se encontra no Display, imediatamente na sua seleção.
